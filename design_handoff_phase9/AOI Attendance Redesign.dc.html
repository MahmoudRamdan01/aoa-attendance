KEY DECISIONS
1Gold is reserved for the primary action, active nav and key figures — the old UI spread 8 accent hues across sections.
2Check-in became a single thumb-reach ring; GPS + face verification run implicitly, security copy moved below the fold.
3Owner payroll: 9 metric tiles + wide table → net-payroll hero, 4 period KPIs, and per-employee cards (net first).
4Hairline borders + soft depth replace heavy shadows; one status color language everywhere; ≥44px targets.
5Owner opens to a live pulse (in / late / absent) + one actionable approvals queue; employees get payslip transparency.
6Trust states live on the check-in ring itself (verifying → success / GPS failure with recovery), offline is a persistent banner.
TRY / TWEAKS

Tap tabs, press تسجيل حضور (verification steps run on the ring), open the bell or "بانتظار قرارك" for the approvals inbox and approve/reject. Tweaks: accent, owner ↔ employee role, texture, offline banner, and a GPS-failure outcome to preview the error state.
